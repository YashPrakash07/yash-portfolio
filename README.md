# Portfolio Website

Welcome to my personal portfolio repository! This website showcases my projects, work experience, skills, and contributions to the open-source community.

It is built with a modern tech stack to ensure high performance, responsiveness, and a great user experience.

## ‚ú® Features

- **Modern UI/UX**: Clean, minimalist design with smooth animations using `framer-motion`.
- **Responsive Design**: Fully responsive layout that looks great on all devices, from mobile to desktop.
- **Dark Mode**: Native dark mode support using `next-themes`.
- **Resume & Experience**: Dedicated sections for work experience, education, and skills.
- **Project Showcase**: Highlighted projects with descriptions, tech stacks, and links.
- **GitHub Contributions**: Integration with GitHub to show contribution activity.
- **Contact**: Easy ways to get in touch, including protected email links to prevent spam.

## üöÄ Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/YashPrakash07/yash-portfolio.git
    cd yash-portfolio
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    # or
    bun install
    ```

3.  **Run the development server**:

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üìù Customization

This portfolio is data-driven. You can update your information by modifying specific files:

- **Profile Data**: Edit `app/data/resume.tsx` to update your name, links, experience, projects, and more.
- **Images**: Place your images (avatar, project screenshots) in the `public` folder and reference them in the data file.
